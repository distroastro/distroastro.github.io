Location Settings Help

Current Location

This section tells you your current system location. This information is read from the file /etc/location.

Location Box

This box displays the following information about your location:

  • Latitude
  • Longitude
  • Elevation
  • Street
  • Area
  • City
  • Region
  • Country
  • Country Code
  • Timezone ID
  • Timezone Name
  • Timezone Abbr
  • UTC Offset
  • DST Offset

This same information can be found by running 'location -a' on the command line.

Show Map

To see a map of your system location, click the button called .

This will open Google Maps and will require an Internet connection.

Location Settings

This sections lets you control if the system and application location information is updated automatically.

Automatically update application location settings

If this is checked, the location settings of each supported application is automatically updated upon user login. For example, upon opening a supported program such as Stellarium, its home location will be your system location.

If it is unchecked, the location settings of supported applications remain unchanged from the last known value.

To update them immediately, click the button called .

This can also be done by running 'autolocation set' on the command line.

Automatically update system location settings

If this is checked, the location settings found in /etc/location is updated automatically when the computer is connected to the Internet.

If it is unchecked, the location settings remain unchanged from the last known value.

To update it immediately, click the button called .

This can also be done by running 'sudo update-location' on the command line.

Manually set system location settings

The current GUI does not have a way to manually set system location. To set it, run the following on the command line:

sudo update-location LAT LONG
